== Description ==

A simple plugin that allows you to customize 8 color swatches underneath the Iris Color Picker. This works for instances of the color picker in themes, plugins, or core. It uses Javascript to send the custom palette to existing Iris objects.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/custom-iris-swatches` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Color Swatches screen to configure the plugin

== Frequently Asked Questions ==

= Does this work for the TinyMCE Editor? =

Nope, not yet.

= Does this work for Divi Builder? =

No, however, Divi Builder has it's own Swatch settings for Iris Color Picker built in!

= Does this work for Visual Composer? =

Yes!

= Does this work for Beaver Builder? =

Not yet!
